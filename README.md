
前端纳米学位街机游戏克隆项目
===============================
## 操作方法
使用上，下，左，右方向键操作人物到达对面，如果游戏中和瓢虫相碰则会返回原点，注意哦。
当成功过河后，可以点击Reset键重新开始，加油！！
